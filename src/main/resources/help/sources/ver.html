Ver información adicional

Para ver información adicional sobre una instancia es necesario disponer de una instancia cargada en la aplicación. Una vez se cumple esta condición puede accederse a la visualización de las siguientes informaciones:

  • Gráfico de duraciones
    • Opción de menú Ver > Duración
    • Utilizando el atajo Ctrl+P
  • Gráfico de fechas de vencimiento
    • Opción de menú Ver > Fechas de vencimiento
    • Utilizando el atajo Ctrl+D
  • Tabla de tareas
    • Opción de menú Ver > Tareas
    • Utilizando el atajo Ctrl+T
