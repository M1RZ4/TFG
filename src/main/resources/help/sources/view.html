View additional information

To view additional information of an instance it's necessary to have an instance loaded in the application. Once this is aquired you can acces the following information:

  • Durations chart
    • Menu option View > Durations
    • Shortcut Ctrl+P
  • Due dates chart
    • Menu option View > Due dates
    • Shortcut Ctrl+D
  • Tasks table
    • Menu option View > Tasks
    • Shortcut Ctrl+T
