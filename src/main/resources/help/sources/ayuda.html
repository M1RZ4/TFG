Acceder a la ayuda

Para acceder a la ayuda (manual de usuario de la aplicación) no es necesario que se dé ninguna condición específica y puede hacerse a través de una de estas opciones:

  • Utilizando la opción de menú Ayuda > Manual de usuario
  • Utilizando el atajo Ctrl+H
  • Utilizando el atajo F1

Una vez accedido a la opción del manual, se desplegará un diálogo donde el usuario podrá acceder a todos los contenidos de la ayuda divididos en dos secciones:

  • Tabla de contenidos
  • Índice

La vista de índice presenta además un cuadro de búsqueda donde se pueden introducir términos para buscar entre los títulos.
