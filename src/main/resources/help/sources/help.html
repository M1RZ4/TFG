Access help system

To acces help system (application's user manual) there's no specific condition required and it can be done by one of these options:

  • Using menu option Help > User Manual
  • Using shortcut Ctrl+H
  • Using shortcut F1

Once you access the option, a dialog is displayed where you can access to all the help contents, divided in two sections:

  • Table of contents
  • Index

Index view has also a search box where you can write different terms to search among the titles.
